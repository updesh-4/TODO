# Fullstack Todo App (React + TypeScript | Node + TypeScript + MongoDB)

This scaffold contains a full backend and frontend setup for the Todo assignment.

## Quick start

### Backend
1. cd backend
2. cp .env.example .env and fill MONGO_URI and other vars
3. npm install
4. npm run dev

### Frontend
1. cd frontend
2. npm install
3. create .env with `VITE_API_URL=http://localhost:4000/api` if needed
4. npm run dev
